# gpx-editor

Visual editor for GPX files

## Running

### Command line

#### Frontend

```powershell
cd frontend
npm run dev
```

#### Backend

```powershell
cd backend
.venv\scripts\Activate.ps1
python manage.py runserver
```
    
### Docker

First, start the Docker daemon.

```powershell
docker compose up -d --build
```
